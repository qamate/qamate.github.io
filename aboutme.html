NBlog is a blogging template completely written from the scratch by Krishnanand Sivraj in Angular2 and Typescript hosted in github (for free of course). Everything in .net starts with 'N', say, for example, NUnit,NInject,NDepend,etc., The same way this blog discusses .Net and related technologies. So the name NBlog. You can also create a website by configuring my blog template for yours from this github url. NBlog is still in pre-alpha as I work on this during my free time. Open for any forks and pull requests. The code is little messy and needs a cleanup. I will improve in my free time.
